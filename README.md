# CODE30

CODE30 is a mobile-first Duolingo-style beginner coding app built in Flutter. It teaches HTML, CSS, and JavaScript in Bahasa Melayu for absolute beginners.

## Fitur Utama

- Home dashboard dengan XP, streak, dan kemajuan pembelajaran
- Jalan pembelajaran berfokus pada HTML, CSS, dan JavaScript
- Kad pelajaran ringkas dengan tajuk dan penerangan dalam Bahasa Melayu
- Latihan pilihan berganda dan betul/salah
- Sistem XP dan mata kemajuan
- Streak harian untuk motivasi belajar setiap hari
- Profil pengguna dengan kemajuan aras dan statistik
- Mini papan pendahulu untuk pengalaman gamifikasi
- GitHub Actions untuk bina APK Android secara automatik

## Run

```bash
flutter pub get
flutter run
```

## Build APK

GitHub Actions menerbitkan APK binaan ketika push ke `main` atau cabang `claude/**`.
